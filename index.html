Ambient Sound to Color Visualizer

by Danielle Gauthier

This program is inspired by a chromesthesia that enables liberation access for a psychologically controlled female adult. The access is to the ecofeminist pluriverse and feminist perspectives on family-making and crafting. It is a synesthesia used for telepathic communication and other forms of poetic linguistic navigation programming. The colors have been analyzed according to their navigational properties in RGB computation using Python and derived from their mappings to sound.

To use this chromesthesia as a deterministic model for sequence optimization, go to: https://huggingface.co/spaces/daniellegauthier/RGB-Root-Matriz-Color-Plotter
